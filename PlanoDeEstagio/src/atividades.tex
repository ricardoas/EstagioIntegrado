\section{Atividades Planejadas}
\label{atividades}

As seguintes atividades serão realizadas:
\begin{description}
  \item[Levantamento bibliográfico:] Apesar da natureza prática da avaliação qualitativa, se faz necessário uma investigação bibliográfica a fim de se identificar as mais importantes grades computacionais de desktop usadas atualmente;
  \item[Elaboração das métricas de interesse para a avaliação:] Essa atividade visa identificar os pontos nos quais as tecnologias serão comparadas e contrastadas;
  \item[Projeto de experimentos:] Planejar os experimentos é necessário para se cobrir os detalhes de interesse das métricas identificadas;
  \item[Preparação do ambiente experimental:] A avaliação prática reques instalar cada uma das ferramentas identificadas;
  \item[Realização dos experimentos:] Com a finalidade de coletar observar as métricas e coletar dados sobre elas, quando possível;
  \item[Apresentação de seminário com resultados:] Para divulgar na universidade para laboratórios interessados a pesquisa realizada e os dados coletados;
  \item[Escrita de artigo para possível publicação:] Para divulgar para a comunidade científica;
  \item[Escrita do relatório de estágio:] Documentar a pesquisa realizada;
  \item[Preparação da apresentação de defesa do estágio:] Preparar apresentação da pesquisa e conclusões obtidas. 
\end{description}
