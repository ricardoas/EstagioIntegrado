\section{Objetivos}
\label{objetivos}

O principal objetivo desse trabalho é comparar de forma qualitativa o uso das principais tecnologias de grades desktops ou oportunistas da atualidade.

Como objetivos específicos espera-se:

\begin{itemize}
  \item Realizar um levantamento bibliográfico sobre as grades desktop ou oportunistas mais usadas atualmente;
  \item Levantar métricas que possam ser usadas nesse estudo comparativo qualitativo;
  \item Comparar qualitativamente, com base nas métricas definidas, as grades identificadas no início do estágio. 
\end{itemize}
