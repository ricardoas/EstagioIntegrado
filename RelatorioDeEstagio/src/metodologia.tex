\chapter{Metodologia}
\label{metodologia}

A idéia deste trabalho é realizar uma análise comparativa a respeito do uso das grades computacionais de desktop mais populares da atualidade. Optou-se por uma abordagem qualitativa para avaliar o uso de cada ferramenta. Inicialmente foi construída uma base de informações sobre as ferramentas estudadas, tais como artigos e documentação das ferramentas. Em um segundo momento, foram escolhidas algumas métricas a serem usadas como base para a comparação entre estas ferramentas. À luz de tais métricas, as ferramentas foram utilizadas e foi realizada uma análise comparativa das ferramentas que refletem o estado da prática nesta área. As métricas levaram em conta as diversas fases do uso das tecnologias como a instalação e o uso da grade em si, como também custos de manutenção e preparação de aplicações para execução na grade, por exemplo.

Dessa forma, pode se dizer que a metodologia deste trabalho se divide nas seguintes etapas:

\begin{itemize}
  \item Formulação do problema e dos procedimentos da pesquisa:
\begin{itemize}
  \item Definição do problema;
  \item Identificação das ferramentas a serem analisadas;
\end{itemize}
  \item Planejamento das métricas avaliadas:
\begin{itemize}
  \item Determinar o que deve ser estudado nas ferramentas;
  \item Identificar aspectos comuns nas ferramentas; 
\end{itemize}
  \item Determinação os cenários de experimentação:
\begin{itemize}
  \item Determinar a seleção de amostras;
  \item Verificar e validar a seleção de amostras no tocante a representar a população; 
\end{itemize}
  \item Realização os experimentos e coleta de dados;
  \item Desenvolvimento de um plano de análise da amostra:
\begin{itemize}
  \item Filtrar dados relevantes;
\end{itemize}
  \item Processar e analisar os dados obtidos gerando uma tabela comparativa das ferramentas usadas.
\end{itemize}

\section{Atividades}

As seguintes atividades foram realizadas:

\begin{description}
  \item[Levantamento bibliográfico:] Apesar da natureza prática da avaliação qualitativa, se faz necessário uma investigação bibliográfica a fim de se identificar as mais importantes grades computacionais de desktop usadas atualmente;
  \item[Elaboração das métricas de interesse para a avaliação:] foram identificados os pontos nos quais as tecnologias serão comparadas e contrastadas;
  \item[Projeto de experimentos:] Planejar os experimentos foi necessário para se cobrir os de\-ta\-lhes de interesse das métricas identificadas;
  \item[Preparação do ambiente experimental:] A avaliação prática requisitou instalar cada uma das ferramentas identificadas;
  \item[Realização dos experimentos:] Com a finalidade de coletar observar as métricas e coletar dados sobre elas, quando possível;
  \item[Escrita do relatório de estágio:] Documentar a pesquisa realizada;
  \item[Preparação da apresentação de defesa do estágio:] Preparar apresentação da pesquisa e conclusões obtidas. 
\end{description}
